Hi!

Old graphics.cfg is gone and replaced with graphics.lua.
However, please, do not edit this file directly.
Otherwise you'll lose your changes after each update.

New method for overriding options is to create (or edit) a file

C:\Users\<YOUR-LOGIN>\Saved Games\DCS\Config\autoexec.cfg

or 

C:\Users\<YOUR-LOGIN>\Saved Games\edModelViewer\Config\autoexec.cfg

(for model viewer)

and place your changes there.
Note:
 - autoexec.cfg settings will override the options set via GUI.
 - Graphic options in autoexec.cfg should be prefixed with options.graphics.
 - file encoding should be UTF-8 without BOM

Example autoexec.cfg:

options.graphics.maxfps = 30
options.graphics.render3D = false
options.graphics.ScreenshotQuality = 75
-- add custom texture path
-- note the slashes!
table.insert(options.graphics.VFSTexturePaths, "c:/Users/<LOGIN>/Saved Games/DCS/MyTextures")

Приветствуем!

Старый файл с настройками графики игры graphics.cfg был заменен на graphics.lua.
Однако, пожалуйста, не редактируйте содержимое файла напрямую,
в противном случае все ваши изменения будут теряться при каждом обновлении игры.

Новый метод замещения настроек заключается в создании (или редактировании) файла

C:\Users\<ИМЯ ПОЛЬЗОВАТЕЛЯ>\Saved Games\DCS\Config\autoexec.cfg

или

C:\Users\<ИМЯ ПОЛЬЗОВАТЕЛЯ>\Saved Games\edModelViewer\Config\autoexec.cfg

(для приложения просмотра 3D моделей (ModelViewer),

и размещении своих настроек в нем.

Обратите внимание:
- настройки в файле autoexec.cfg замещают настройки, заданные через графический интерфейс игры.
- опции настроек графики в файле autoexec.cfg должны содержать префикс 'options.graphics'.
- файл autoexec.cfg должен быть сохранен в кодировке UTF-8 (без BOM).

Примеры описания настроек в autoexec.cfg:

options.graphics.maxfps = 30
options.graphics.render3D = false
options.graphics.ScreenshotQuality = 75
-- описание пути для текстур пользователя
-- обратите внимание на символ косой черты '/' !
table.insert(options.graphics.VFSTexturePaths, "c:/Users/<ИМЯ ПОЛЬЗОВАТЕЛЯ>/Saved Games/DCS/MyTextures")
