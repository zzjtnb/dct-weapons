To install the example:

1. Create a folder for your custom resource mods
   C:\Users\<your windows username>\Saved Games\DCS[.variant_suffix]\Mods\resource\
2. Copy SoundModExample folder to it.
3. Start DSC and run the example_soundmod.miz mission
4. Enjoy a mighty Tunguska sounding line something completely different. ;)
